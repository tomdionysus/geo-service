GeoService

Geographic Information Service

GeoService is a PROTOTYPE. Please do not use in production Applications.

v0.0.1

Introduction

GeoService is a fast and lightweight service intended to provide geographic data, including:

  • UN M.49 Regions
  • ISO 3166-1 Countries
  • ISO 3166-2 Country Subdivisions
  • ISO 4217 Currency Codes
  • ISO 639 Language Codes
  • IP to Country Mapping

GeoService also provides this data in a relational sense, i.e. it is possible to query, for example:

  • All countries using the Euro
  • All states in the USA
  • All regions and their countries in Africa
  • All countries with Portuguese as an official language
  • The country for the IP address 212.67.202.4

JSON API

Use the resources detailed here as a RESTful JSON resource, i.e. the GET verb and Accept: application/json

General API

Most resources can be accessed as follows:

  • /:resource_name - Returns an array of all entities.
  • /:resource_name/:code - Returns a single entity.
  • /:resource_name?key=value&key2=value2... - Returns an array of all entities where all keys match all values .
  • /:resource_name?include=relation,relation2... - Returns an array of all entities, including relations as subobjects.
  • /:resource_name/:code/?include=relation,relation2... - Returns a single entity, including relations as subobjects.
  • /:resource_name?exclude=field,field2... - Returns an array of all entities, excluding specific fields.
  • /:resource_name/:code?exclude=field,field2... - Returns a single entity, excluding specific fields.

Resources

Regions

Resource: /regions

Regions represent geographical areas, e.g. continents and major landmasses. Regions can be nested.

Relations available using include:

  • countries - /countries?include=currency
  • regions - /regions?include=subregions

Countries

Resource: /countries

Countries respresent geopolitical entities.

Relations available using include:

  • currency - /countries?include=currency
  • regions - /countries?include=regions
  • subdivisions - /countries?include=subdivisions
  • languages - /languages?include=languages

Subresources:

  • subdivision - /countries/:country_code_alpha_2/:subdivsion_code e.g. /countries/AU/AU-NSW

Subdivisions

Resource: /subdivisions/:subdivision_code e.g. /subdivisions/ENG

Subdivisions represent subdivisions within a country, e.g. states, provinces, protectorates and the like. Subdivisions can be nested.

Relations available using include:

  • subdivisions - e.g. /subdivisions/ENG?include=subdivision

Currencies

Resource: /currencies

Currencies represent the currencies in use globally.

Relations available using include:

  • countries - /currencies?include=countries

Languages

Resource: /languages

Currencies represent the languages in use globally.

Relations available using include:

  • countries - /languages?include=countries

GeoIP

Resource: /geoip/:ip_address

Get the ISO3166-1 Country Code for a given IP4 Address in xxx.xxx.xxx.xxx format.

No relations are available on this resource.
